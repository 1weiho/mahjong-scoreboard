# Mahjong Scoreboard

A web-based scoreboard for Taiwanese Mahjong that enables users to save data on their browser and export it for further use.

## Technologies

- Vite
- React
- Typescript
- Tailwind CSS

## Setup Project

```bash
# install dependencies
npm install

# run in localhost:5173/mahjong-scoreboard
npm run dev
```
